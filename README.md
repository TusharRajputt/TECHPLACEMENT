# Quote of the Day

## Description
ðŸš€ Welcome to the "Quote of the Day" project! This is a dynamic web application built as a full-stack project using React for the frontend and Node.js for the backend. It fetches random inspirational quotes, allows users to add comments, and more.

## Features
- Fetch a random quote from an external API.
- Display the quote and its author.
- Add and view comments for each quote.
- Thumbs-up functionality for quotes.
- Responsive design with sleek UI.

## Technologies Used
- **Frontend:** React, CSS
- **Backend:** Node.js, Express
- **HTTP Client:** Axios

## Installation
1. **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/quote-of-the-day.git
    cd quote-of-the-day
    ```

2. **Install dependencies:**
    - For the backend:
        ```bash
        cd backend
        npm install
        ```
    - For the frontend:
        ```bash
        cd ../frontend
        npm install
        ```

3. **Run the application:**
    - For the backend:
        ```bash
        cd backend
        node server.js
        ```
    - For the frontend:
        ```bash
        cd ../frontend
        npm start
        ```

## Usage
1. **Start the server:**
    ```bash
    cd backend
    node server.js
    ```

2. **Start the frontend:**
    ```bash
    cd ../frontend
    npm start
    ```

3. Open your web browser and navigate to `http://localhost:3000` to view the application.
